vim molokai 配色设置



1. 下载文件 molokai.vim
                



2. 拷贝 molokai.vim 文件

	将 molokai.vim 文件放置在: /usr/share/vim/vim74/colors/
	
	#进入文件目录, 拷贝文件
	cp molokai.vim /usr/share/vim/vim74/colors/
	 
                



3. 设置 vi 指向 vim 的软连全局使用 vim 功能
	
	#备份 vi 软连
	mv /usr/bin/vi /usr/bin/vi.bak
	#重写 vi -> /usr/bin/vim
	ln -s /usr/bin/vim /usr/bin/vi
	
                



4. 配置全局的vimrc文件对所有用户的vim配置生效

	(一般不设置全局)全局vimrc路径: /etc/vimrc
	
	#编辑 /etc/vimrc
	vi /etc/vimrc
	 
                



5. 用户自定义vim molokai配色设置

	.vimrc 文件路径: ～/.vimrc 
	(默认第一次使用vim命令后文件才会创建,初次创建用户该文件不存在新建即可)
	
	#编辑 ~/.vimrc
	vi ~/.vimrc
	
	打开文件后加入如下配置
	
	set nocp
	set ru
	syntax on
	set number
	set tabstop=4
	set autoindent
	set shiftwidth=4
	filetype on
	set showmatch
	set matchtime=5
	set cindent

	let g:molokai_original = 1
	let g:rehash256 = 1
	set t_Co=256
	colorscheme molokai
	
                



6. 配置命令别名
	
	#编辑 ~/.bashrc
	vi ~/.bashrc
	
    
	alias rm='rm -i'
	alias cp='cp -i'
	alias mv='mv -i'
